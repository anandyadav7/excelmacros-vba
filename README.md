# excelmacros.net — VBA source

This is the public, plain-text source of every macro shipped on
[excelmacros.net](https://excelmacros.net).

Each `.bas` file is a standalone VBA module. To run one:

1. Open Excel, press **Alt + F11** (Windows) or **Option + F11** (Mac).
2. **File → Import File…** and pick the `.bas`.
3. Close the editor and press **Alt + F8** to run the macro.

Or just download the prebuilt `.xlsm` from the matching tool page
on excelmacros.net and skip the import step.

This repo exists for transparency: every line of VBA we run is here for you to read.
The website source itself lives in a separate (private) repo.
